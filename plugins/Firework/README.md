Install the extension and then add the `firework` compiler plugin to your build config.

Learn more at https://github.com/outercloudstudio/Firework

```json
{
    "plugins": [
        [
            "comMojangRewrite",
            {
                "packName": "BridgeTest"
            }
        ],
        [
            "firework",
            {
                "delayChannels": 3
            }
        ]
    ]
}

```
